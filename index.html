Asset APIs

REST API

The Asset APIs provide comprehensive endpoints for managing assets, permissions, and asset-related operations. These APIs allow you to create, retrieve, update, and delete asset instances, as well as manage asset permissions and check access control.

Key Features:

  • Asset instance management
  • Permission assignment and verification
  • Asset queries and filters
  • Batch operations

View Asset APIs Documentation

Attribute APIs

REST API

The Attribute APIs enable you to manage custom attributes and attribute definitions for assets. These APIs provide functionality for creating attribute groups, defining attribute properties, and assigning attribute values to specific assets.

Key Features:

  • Attribute group management
  • Attribute definition creation
  • Batch value operations
  • Flexible data types
